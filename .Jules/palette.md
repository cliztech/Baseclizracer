# Palette's Journal - Critical Learnings

## 2025-10-26 - Project Initialization
**Learning:** This is a legacy-style "Outrun" racing game. The codebase uses global variables and embedded JS for simplicity.
**Action:** Be extremely careful with variable scope and avoid modern ES module syntax unless I'm sure it's supported or transpilable. Focus on semantic HTML and ARIA where possible in the HTML files.
